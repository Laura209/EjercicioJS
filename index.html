Calculadora de promedio

El siguiente programa realizará el cálculo de tres notas parciales, un examen parcial y un trabajo final de un estudiante, para obtener el promedio y verificar si aprobó o reprobó.

Las notas que se van a ingresar deben estar en un rango de 0 a 50.
